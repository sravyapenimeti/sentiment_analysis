# sentiment_analysis

This project predicts whether a customer review is Positive or Negative using Machine Learning.

## Features

- Text Cleaning
- TF-IDF Vectorization
- Logistic Regression
- Streamlit Interface

## Accuracy

89%

## Technologies

- Python
- Streamlit
- Scikit-learn
- NLTK
